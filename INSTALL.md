# 📦 ESCANOR BOT V3 — قسم المطورين فقط

## 📋 المتطلبات
- Node.js 18+ 
- npm أو yarn
- رقم واتساب فعّال

---

## ⚡ التثبيت السريع

### 1️⃣ فتح المجلد
```bash
cd escanor-bot-v1
```

### 2️⃣ تثبيت المكتبات
```bash
npm install
```

### 3️⃣ إعداد البيانات
افتح `config.js` وعدّل:
```js
ownerNumber: ['رقمك بدون +'],   // مثال: '201234567890'
botOwner: 'اسمك',
botName: 'اسم البوت',
```

### 4️⃣ تشغيل البوت
```bash
npm start
```

---

## 📱 ربط الواتساب

1. شغّل البوت بـ `npm start`
2. سيظهر لك: `أدخل رقم الهاتف (مع كود الدولة بدون +):`
3. اكتب رقمك مثل: `201234567890`
4. ستحصل على **كود الربط** مثل: `ESCANOR1`
5. افتح واتساب على هاتفك
6. اذهب إلى: **الإعدادات > الأجهزة المرتبطة > ربط جهاز**
7. اختر "ربط بالرقم" وأدخل الكود

## ⚙️ الإعدادات المتقدمة

### تغيير البادئة
في `config.js`:
```js
prefix: '.', // يمكن تغييره إلى / أو ! أو غيره
```

### الحماية
```js
antiLink: true,    // حماية من الروابط
antiSpam: true,    // حماية من السبام
antiFlood: true,   // حماية من الفلود
floodLimit: 5,     // عدد الرسائل المسموح
floodTime: 5000,   // الوقت بالميلي ثانية
```

### الرتب (18 رتبة)
```
0. جندي → 1. عريف → 2. رقيب → 3. رقيب أول
4. مساعد → 5. ملازم → 6. ملازم أول → 7. نقيب
8. رائد → 9. مقدم → 10. عقيد → 11. عميد
12. لواء → 13. فريق → 14. فريق أول → 15. مشير
16. مساعد وزير → 17. وزير
```

## 🐞 المشاكل الشائعة

### مشكلة: `Cannot find package`
```bash
npm install
```

### مشكلة: `Session expired`
احذف مجلد `session/` وشغّل مجدداً

### مشكلة: الكود لا يظهر
- تأكد أن الرقم صحيح (بدون + وبكود الدولة)
- انتظر 30 ثانية وأعد المحاولة
